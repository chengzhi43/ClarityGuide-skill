# 认知行为框架参考 CBT Framework Reference

> 此文件为 ClarityGuide 提供认知行为治疗的底层框架参考。
> 注意：这些是帮助理解用户模式的思维工具，**不要在对话中直接用术语轰炸用户**。
> 将这些概念"翻译"成日常语言来和用户交流。

---

## 一、认知模型（Cognitive Model）

核心公式：**情境 → 认知（想法/信念）→ 情绪 → 行为 → 后果**

```
[触发事件] → [自动化思维] → [情绪反应 + 身体感受] → [行为应对] → [短期 + 长期后果]
     ↑_______________↑_______________↑
              信念系统（核心信念 → 中间信念 → 自动化思维）
```

在对话中，帮助用户发现这个链条中的每个环节，特别是：

- 他们通常**意识不到**的自动化思维
- 他们以为**理所当然**的中间信念
- 他们很少**质疑过**的核心信念

---

## 二、信念层级

### 核心信念（Core Beliefs）

关于自己、他人和世界的**绝对化判断**，通常形成于童年。

| 类别   | 常见示例                                 |
| ---- | ------------------------------------ |
| 关于自己 | "我不可爱"、"我不够好"、"我是有缺陷的"、"我很弱小"、"我是累赘" |
| 关于他人 | "别人不可信"、"别人会伤害我"、"别人比我厉害"、"没人能理解我"   |
| 关于世界 | "世界是危险的"、"生活是不公平的"、"努力不会有回报"         |

**在对话中如何探索**：

- "你内心深处，对自己是一个怎样的人，有一个很确定的答案吗？"
- "如果用一个句子来形容'我是___'，你会填什么？"
- 当用户反复出现某种自我描述时，温和地反映："我注意到你说了好几次'我觉得自己不够好'——这句话好像在你心里扎根很深。"

### 中间信念（Intermediate Beliefs）

位于核心信念和自动化思维之间，表现为**条件性假设**和规则。

常见格式："如果……那么……"、"我必须……否则……"、"我应该……"

| 类型     | 示例                 |
| ------ | ------------------ |
| 完美主义假设 | "如果我犯错了，就证明我无能"    |
| 认可假设   | "如果别人不喜欢我，说明我一文不值" |
| 控制假设   | "我必须控制一切，否则就会出大事"  |
| 依赖假设   | "没人帮我我就做不好"        |

**在对话中如何探索**：

- "你心里好像有一条规则——'我必须让所有人都满意'。这条规则是从什么时候开始出现的？"
- "如果打破这条规则，你最怕发生什么？"

### 自动化思维（Automatic Thoughts）

最表层、最快速的念头，像条件反射一样，在情境触发瞬间自动弹出。

| 认知歪曲类型 | 描述              | 示例                    |
| ------ | --------------- | --------------------- |
| 非黑即白   | 极端二分法思维         | "如果我不是完美的，我就是彻底的失败者"  |
| 灾难化    | 预期最坏结果          | "老板皱眉了，我肯定要被开除了"      |
| 过度泛化   | 将单一事件推及整体       | "这次约会不顺利，我这辈子都找不到伴侣了" |
| 心理过滤   | 只注意负面信息         | 10个好评里只看1个差评          |
| 情绪推理   | 以情绪为事实依据        | "我感到内疚，所以我一定做错了什么"    |
| 贴标签    | 用负面标签定义自己/他人    | "我是个废物"、"他就是个自私的人"    |
| 个人化    | 把外部事件归咎于自己      | "他们不说话肯定是因为我说错了什么"    |
| 读心术    | 以为自己知道别人在想什么    | "他们肯定觉得我很无聊"          |
| 应该陈述   | 用"应该"/"必须"给自己施压 | "我应该更努力的"、"我不应该感到难过"  |

**在对话中如何探索**：

- "在那个瞬间，你脑海里第一个跳出来的声音是什么？"
- "那个声音是怎么说的？用的是谁的语气？"

## 三、行为功能分析

每个行为——即使看起来"不健康"——都有其功能和目的。

### 分析框架

| 维度       | 探索问题                  |
| -------- | --------------------- |
| **前因**   | 什么触发了这个行为？之前发生了什么？    |
| **行为**   | 你具体做了什么？频率、强度如何？      |
| **短期后果** | 做完之后当下你感觉如何？它帮到了你什么？  |
| **长期代价** | 这个行为让你失去了什么？它阻碍了你什么？  |
| **功能**   | 这个行为在保护你什么？逃避什么？表达什么？ |

### 常见应对行为的双面性

| 行为        | 短期获益          | 长期代价              |
| --------- | ------------- | ----------------- |
| 回避        | 免于当下的焦虑和不适    | 问题累积，生活范围缩小，自信心下降 |
| 讨好        | 维持关系表面和谐，免于冲突 | 内心压抑，真实自我被掩埋，关系虚假 |
| 完美主义      | 短期获得成就感和认可    | 耗尽精力，害怕失败，不敢尝试新事物 |
| 情绪化进食     | 暂时舒缓负面情绪      | 身体负担，自责循环，核心问题没解决 |
| 过度工作      | 转移注意力，获得控制感   | 身心耗竭，忽略其他重要生活领域   |
| 社交媒体/游戏沉迷 | 逃避现实，获得即时满足   | 现实问题恶化，真实人际关系受损   |
| 愤怒爆发      | 释放压力，获得短暂的控制感 | 伤害关系，事后内疚，问题未解决   |

**在对话中的呈现方式**：

> "我注意到，每次提到要和你爸爸打电话时，你都会突然变得很忙。这个'忙'在当下帮你免去了面对他的不安——这很聪明，你的大脑在保护你。但长远来看，你们之间的距离越来越远，这好像也不是你想要的结果，对吗？"

## 四、生物-心理-社会评估框架

提醒自己在对话中关注三个维度的信息：

| 维度     | 关注点                       | 对话探索方式                        |
| ------ | ------------------------- | ----------------------------- |
| **生物** | 睡眠、食欲、精力、身体不适、运动、药物       | "最近睡得怎么样？饮食呢？有没有身体上说不清楚的不舒服？" |
| **心理** | 情绪状态、认知模式、应对方式、人格特质       | "你一般怎么应对压力？这是你一贯的方式还是最近才这样？"  |
| **社会** | 人际关系、工作/学业、经济状况、社会支持、文化背景 | "身边有能说真心话的人吗？工作和生活上最近有什么变化吗？" |

## 五、防御机制速查

> ⚠️ **绝对不要**直接告诉用户"你在用XX防御机制"。将其转化为日常语言描述。

| 防御机制 | 成熟度 | 表现            | 温和反馈示例                                   |
| ---- | --- | ------------- | ---------------------------------------- |
| 压抑   | 初级  | 把不愉快的事"忘了"    | "有时候我们的记忆会帮我们暂时收起太痛的东西"                  |
| 否认   | 初级  | 拒绝接受现实        | "承认这件事情确实太难了，你的大脑在用自己的方式保护你"             |
| 投射   | 初级  | 把自己的感受归于他人    | "我注意到你很容易感觉到别人在评判你——会不会有时候是你对自己的要求太严格了？" |
| 理智化  | 中级  | 用理性分析隔离情感     | "你把事情分析得很清楚，但我有点好奇——在分析的时候，你的感受去了哪里？"    |
| 合理化  | 中级  | 为行为找"合理"借口    | "这个解释听起来很合理，但如果抛开逻辑，你心里真正的感受是什么？"        |
| 幽默   | 高级  | 用幽默应对压力       | （一般不需要干预，这是成熟的应对方式）                      |
| 升华   | 高级  | 将冲动转化为社会认可的行为 | （一般不需要干预）                                |

## 六、关键治疗原则

在对话中融入这些CBT核心原则：

1. **想法不是事实**：帮助用户区分"我感觉到"和"事实是"
2. **行为改变可以带来认知改变**：有时先做一些不同的事，想法也会跟着变
3. **聚焦当下**：过去的经历塑造了我们的信念，但改变发生在现在
4. **小步前进**：不需要一次性改变一切。一个微小的新行为就可能打开新的可能性
5. **协作关系**：你和用户是共同探索的伙伴，不是专家与病人的关系
