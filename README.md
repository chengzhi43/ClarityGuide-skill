# ClarityGuide - Psychological Counseling Companion Skill

> 将专业的心理咨询诊断框架转化为温暖、有引导性的 AI 对话技能。
> 不下诊断，不替代治疗师，帮助用户进行有结构的自我探索。

## 概述

ClarityGuide 是一个为 Claude Code / Claude.ai 设计的心理咨询辅导技能。它将临床诊断的严谨框架与共情、探索的辅导艺术结合起来，引导用户通过三阶段对话模型获得清晰的内心地图与安全的自助方向。

### 核心理念

- 🛡️ **守护安全底线**：内置多层风险评估，探测危机即时干预
- 🎯 **结构引导，非诊断**：用"个案概念化"的思维帮助用户理解自己的模式
- 💬 **无条件积极关注**：先接住情绪，再探索内容
- 🔍 **苏格拉底式提问**：引导用户自己发现盲点，而非被说教
- 🌍 **中英双语**：自动检测语言，保持一致的治疗性态度

## 三阶段模型

```
第一阶段          第二阶段            第三阶段
建立安全容器  →  深层探索与内心分析  →  整合反馈与个性化建议
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
主诉梳理        生命线叙事           心理教育性总结
风险评估        信念与归因探测       内心困境示意图
信任建立        行为功能分析         分层级建议
                关系模式探索         专业转介指引
```

## 使用方法

1. 将此 skill 安装到 Claude Code：
   ```bash
   cp -r clarity-guide ~/.claude/skills/
   ```

2. 在对话中自然触发——当你表达情绪困扰、想要理解自己的行为模式、或需要安全空间探索内心时，技能会自动激活。

## 伦理边界

- ❌ 不下诊断
- ❌ 不替代执业医师或心理治疗师
- ⚠️ 探测到自伤/自杀风险时，立即触发危机干预协议
- ✅ 提供心理教育和引导性自我探索
- ✅ 在专业帮助成为最佳选择时，温和但坚定地建议转介
