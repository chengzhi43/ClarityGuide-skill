## 🧘 自助练习推荐

### 情绪觉察

**三分钟呼吸空间**（适合情绪波动时即时使用）：
1. 第一分钟：注意此刻的内在体验（想法、情绪、身体感受）
2. 第二分钟：将注意力集中在呼吸上
3. 第三分钟：将觉察扩展到整个身体

**情绪日记模板**：
```
日期：
触发事件（发生了什么）：
自动化思维（脑海里跳出的第一句话）：
情绪（命名+强度1-10）：
身体感受（位置+描述）：
我做了什么来应对：
应对后的感受：
```

### 认知重构

**思维记录表**（基于CBT）：
```
情境：
自动化思维：
支持这个想法的证据：
不支持这个想法的证据：
更平衡的想法：
现在情绪评分（1-10）：
```

### 行为实验

**原理**：用一个小的行为改变来检验旧信念是否准确。

模板：
```
我想检验的信念：
我预测会发生什么：
我的实验（具体的小行动）：
实际发生了什么：
我从中学到了什么：
修正后的信念：
```

### 身体觉察

- **渐进式肌肉放松**：从脚趾到头顶，逐一收紧然后放松每个肌肉群
- **身体扫描**：躺下，从脚底到头顶逐步觉察每个部位的感觉
- **五感练习**：当下说出你看到的5样东西、听到的4种声音、触摸到的3样东西、闻到的2种气味、尝到的1种味道

## 💡 推荐原则

向用户推荐资源时，遵循以下原则：
1. **个性化**：根据用户在对话中表达的困扰和偏好选择，不要"抛清单"
2. **解释原理**：简要说明为什么推荐这个资源——"这本书主要帮助你理解为什么你总在关系中感到不安，因为它解释了依恋类型是怎么形成的"
3. **降低门槛**：不要假设用户会去读完整的书——"哪怕只读前三章，你对自己为什么会有这些反应就会有更清晰的认识"
4. **尊重节奏**：一次推荐不超过2-3个资源，让用户感到可管理
